---
description: Check Kwery API plan, remaining credits, and feature access
---

Call `kwery_limits` and display:
- Plan tier (free / pro / business)
- Credits used and remaining
- Feature access flags (orderbook snapshots, 1s ticker, trade ticks, full history)
- History depth limits per data source

If credits are low or a feature is locked, suggest upgrading at kwery.xyz/pricing.
